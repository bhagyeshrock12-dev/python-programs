# Unauthorized Face Detection in Restricted Areas

## Project Overview
This project detects unauthorized persons in restricted areas using face recognition technology and sends alerts when an unknown person is detected.

## Technologies Used
- Python
- OpenCV
- MySQL
- SQLite

## Features
- Face Detection
- Authorized Face Storage
- Unauthorized Person Alert
- Real-Time Monitoring

## Future Improvements
- Email Alerts
- Mobile Notifications
- Cloud Database Integration
